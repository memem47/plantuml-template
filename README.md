# PlantUML Template

This repository provides simple PlantUML examples and a reusable folder/workflow layout
to generate SVG diagrams automatically on push.

## Folder structure

```text
docs/
  uml/                 # Source diagrams (*.puml)
    component_overview.puml
    sequence_execute.puml
    class_model.puml
    activity_flow.puml
    state_machine.puml
    usecase_overview.puml
    deployment_topology.puml
  images/
    uml/               # Rendered outputs (*.svg) generated by GitHub Actions
.github/
  workflows/
    plantuml.yml       # CI to render diagrams and commit SVG outputs
```

## How it works (GitHub Actions)

The workflow `.github/workflows/plantuml.yml` does the following:

- Runs on push to main when docs/uml/*.puml changes (and optionally README/workflow changes).
- Installs PlantUML + Graphviz on the runner.
- Renders docs/uml/*.puml into SVG files under docs/images/uml/.
- Commits and pushes the generated SVGs back to the repository.
